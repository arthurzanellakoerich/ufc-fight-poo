# 🥊 Ultra Combat

Projeto simples desenvolvido em Java utilizando Programação Orientada a Objetos (POO), simulando lutas entre lutadores com base em categoria de peso.

---

## 🚀 Funcionalidades

- Cadastro de lutadores
- Definição automática de categoria por peso
- Simulação de lutas
- Sistema de vitória, derrota e empate
- Atualização de cartel dos lutadores

---

## 🧠 Conceitos aplicados

- Programação Orientada a Objetos (POO)
- Encapsulamento
- Classes e Objetos
- Métodos
- Arrays de objetos
- Uso de Random para simulação

---

## 📂 Estrutura do projeto

- `Lutador.java` → Classe que representa um lutador
- `Luta.java` → Classe responsável pela lógica da luta
- `Main` → Classe principal (main)
